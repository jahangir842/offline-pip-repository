Install a Python Package

Open a terminal and run the following command to install a pip package:

pip install --no-index --find-links=http://192.168.1.109 --trusted-host 192.168.1.109 [package-name]

Example:

pip install --no-index --find-links=http://192.168.1.109 --trusted-host 192.168.1.109 numpy

To Install pip on offline computer, Miniconda can be installed. Download Miniconda for Ubuntu or Windows
