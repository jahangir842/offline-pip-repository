Overview

The pip configuration file (`pip.conf` for Linux/Mac or `pip.ini` for Windows) is used to customize your pip installation. This file allows you to specify settings such as the index URL and trusted hosts for your package installations.

Edit pip.conf on Linux

On Linux, the `pip.conf` file is located in one of the following directories:

  • Global configuration: /etc/pip.conf
  • User configuration: ~/.pip/pip.conf
  • Virtual environment configuration: $VIRTUAL_ENV/pip.conf

To edit the configuration file, you can use any text editor. For example, using nano:

sudo nano /etc/pip.conf

Add or edit the following content in the file to specify the index URL and trusted hosts:

[global]
index-url = http://192.168.1.109/simple
trusted-host = 192.168.1.109

Save and close the file. Your pip installations will now use the local repository for package downloads.

Edit pip.ini on Windows

On Windows, the pip configuration file is called pip.ini. It can be located in one of the following directories:

  • Global configuration: C:\ProgramData\pip\pip.ini
  • User configuration: C:\Users\YourUsername\AppData\Roaming\pip\pip.ini
  • Virtual environment configuration: path\to\virtualenv\pip.ini

To edit the configuration file on Windows, open a text editor like Notepad and create or edit the pip.ini file. Add the following content:

[global]
index-url = http://192.168.1.109/simple
trusted-host = 192.168.1.109

Save the file. Your pip installations will now use the local repository for package downloads.

Test Your Configuration

To test if the configuration is working correctly, run the following command to install a package using the local repository:

pip install --no-index --find-links=http://192.168.1.109 [package-name]

If the package is available, pip will install it from the specified repository. If it's not available, you can contact your IT team for further assistance.
